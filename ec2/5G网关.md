# OneBox 5G 网关 PG03 产品规格书 (V1.1)

## 产品介绍

PG03是OneBox 5G网关系列产品，采用高性能处理器，支持5G SA/NSA，提供2个千兆以太网口，支持eSIM，支持OneCyber平台接入，提供网关管理功能，满足5G无线接入、设备联网、远程控制、高可靠传输等需求，适用于工业组网、室外监控、电力数采等应用场景。

## 应用拓扑

- PG03 ↔ OneCyber
- 电力保护装置
- 工业设备
- 光伏数据采集摄像头
- 室外监控
- 电力数据采集电力保护
- 工业组网

## 产品特点

### 高速外设接口
- ✓ 2×千兆以太网口
- ✓ 1×RS232+1×RS485串行接口

### 高可靠性
- ✓ 支持eSIM
- ✓ 支持黑白名单配置

### 设备管理
- ✓ 支持IPSec、L2TP等主流VPN协议
- ✓ 支持桥接模式、后路由、防火墙功能

### 网络安全
- ✓ 支持5G SA组网
- ✓ 支持5G LAN、高精度授时
- ✓ 支持IPv4、IPv4&IPv6双栈5G通信

### 外观使用
- ✓ 金属外壳，防护等级 IP40
- ✓ 支持平置、导轨式安装

### 设备管理
- ✓ OneCyber平台管理
- ✓ 网关固件升级
